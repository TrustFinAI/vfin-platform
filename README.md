# AI Financial CPA Dashboard

This project is a sophisticated web application designed to serve as a virtual AI financial CPA for small business owners. It allows users to upload their core financial statements (Balance Sheet, Income Statement, Cash Flow) and receive a comprehensive, AI-powered analysis of their business's financial health.

---

## Project Status: V2 - Production Architecture

As of the latest version, this application has been architected for a production environment.

*   **Purpose:** To provide a secure, scalable, and reliable financial analysis service.
*   **Technology:** The application uses a modern client-server model.
    *   **Frontend:** A React/TypeScript single-page application.
    *   **Backend:** A Node.js/Express server running on Google Cloud Run.
    *   **Database:** A managed PostgreSQL database on Google Cloud SQL.
*   **Security:** The backend manages all API keys and communication with AI services. Data is persisted in a secure, centralized database.

---

## Core Features

*   **Secure Authentication:** A client-side user registration and login system (to be replaced with server-side authentication).
*   **Multi-Client Architecture:** All data is securely stored per client in a centralized database.
*   **Guided Onboarding:** New users are guided through a one-time profile setup to provide business context (Industry, Model, Goals) for tailored AI analysis.
*   **Three-Statement Upload:** A clear, intuitive interface for uploading the Balance Sheet, Income Statement, and Cash Flow Statement.
*   **AI-Powered Financial Parsing:** Utilizes the Gemini API (`gemini-2.5-pro`) via a secure backend to parse raw text from financial statements into structured JSON data.
*   **Comprehensive KPI Dashboard:** Displays a wide range of key financial metrics in two sections: "Snapshot Financial Data" and "Key Performance Indicators (KPIs)".
*   **AI Financial Health Score:** A top-level summary card that provides an AI-generated score (0-100), rating, and a breakdown of financial strengths and weaknesses.
*   **AI Analysis & Insights:** An "Executive Summary" and "CPA Actionable Recommendations" generated by Gemini, tailored to the client's profile.
*   **Top 5 Expenses Breakdown:** A visual component showing the main expense categories and their percentage of total costs.
*   **Interactive KPI Explanations:** Users can click on any KPI to receive a simple, AI-generated explanation of that metric (`gemini-2.5-flash`).
*   **Multi-Period Analysis:** The ability to upload and switch between different financial periods, enabling comparative analysis (e.g., Revenue Growth).
